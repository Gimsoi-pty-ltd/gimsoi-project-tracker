# Gimsoi Frontend

## Setup

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Start the development server:
   ```powershell
   npm run dev
   ```

## Structure
- `src/components/`: Reusable UI components
- `src/pages/`: App pages (Dashboard, Login, Kanban, Reports)
- `src/assets/`: Static assets (e.g., logo)

## Tech Stack
- React (Vite)
- Tailwind CSS
- React Router (to be added)
